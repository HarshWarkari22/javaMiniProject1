Java Library Mini Project

This is a basic Core Java mini project demonstrating Object-Oriented Programming concepts.

## Concepts Used
- Classes and Objects
- Default Constructor
- Static Block
- Instance Block
- toString Method
- Return Type Methods
- Package Structure

## Project Structure
src/com/library
- Book.java
- BookDetails.java
- LibraryShelf.java

## Purpose
This project was created to practice Java fundamentals and understand class lifecycle behavior.
